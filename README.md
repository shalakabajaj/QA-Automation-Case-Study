# QA Automation Engineering Case Study

This project contains 3 parts:
1. Fixing flaky Playwright tests
2. Designing a simple automation framework
3. API + UI integrated test for project creation

Tech Used:
- Python
- Pytest
- Playwright
- Requests (API Testing)

Folder Structure:
- config/ → environment and test data
- pages/ → Page Object classes
- tests/ → API, UI, and integration tests
- utils/ → helper functions
